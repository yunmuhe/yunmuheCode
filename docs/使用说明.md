## 使用功能

### 生成姓名页面

1. **输入描述**：在输入框中输入角色描述，例如：
   - "一个勇敢的战士，充满正义感"
   - "温柔善良的女孩，喜欢读书"
   - "聪明机智的少年，擅长科技"

2. **选择参数**：
   - **文化风格**：现代中文、传统中文
   - **性别**：男性、女性、中性
   - **年龄**：儿童、成年人、长者
   - **数量**：1-10个
   - **API**：paiou（推荐）、aistudio、aliyun、baishan

3. **生成姓名**：点击发送按钮，等待AI生成

4. **查看结果**：
   - 姓名及其含义
   - 文化风格标签
   - 姓名特征（笔画、音韵等）

5. **收藏功能**：点击爱心图标收藏喜欢的姓名

### 收藏页面

- 查看所有收藏的姓名

- 批量管理收藏
- 点击"去生成名字"按钮跳转到生成页面

### 历史记录页面

- 查看生成历史
- 搜索历史记录
- 分页浏览

### 设置页面

- 选择默认API
- 主题设置
- 其他个性化配置

## 常见问题

### Q1: 前端无法连接后端？
**A**: 检查：
1. 后端服务是否运行（访问 http://localhost:5000/health）
2. 前端配置文件 `.env` 中的 API 地址是否正确
3. 防火墙是否阻止了5000端口

### Q2: 生成失败或返回错误？
**A**: 检查：
1. 描述是否太短（至少5个字符）
2. 选择的API是否可用
3. 查看后端控制台的错误信息

### Q3: 生成速度慢？
**A**:
1. 切换到 paiou API（速度最快）
2. 减少生成数量
3. 启用缓存功能

### Q4: 收藏功能不工作？
**A**:
1. 确保后端服务正常运行
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存后重试
